# Spring Boot Demo Project

A simple demo application built with **Spring Boot**, demonstrating a classic layered architecture using:

- **Spring Web**
- **Spring Data JPA**
- **MySQL Database**
- **Thymeleaf Template Engine**
- **Entity / Repository / Service / Controller Architecture**

This project is designed as a starter template for building full-stack web applications with Spring Boot.
